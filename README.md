# LifeTime_Q1
Lifetime

A prototype exploring age as both progression and a timer.

Core Concept:
The player progresses through life stages while attempting to complete levels before dying of old age.

Features:
- Age-based progression
- Child → Adult → Elder stages
- Combat unlocks with age
- Multiple levels
- Browser-playable WebGL build
